Puissance X avec Interface

Jeu de plateau avec IA et interface graphique en Java

Objectifs

    Concevoir un jeu de plateau de type Puissance X en respectant les règles de base du gameplay.

    Développer plusieurs intelligences artificielles capables d’adapter leurs stratégies en fonction du comportement ou du niveau du joueur adverse.

    Réaliser une interface graphique réactive, ergonomique et intuitive, permettant une prise en main facile par les utilisateurs.

Technologies utilisées

  • Java
  • Algorithmes IA
  • Interface graphique Boardifier

Déroulement

  • Analyse du besoin : compréhension des règles du jeu Puissance X et des attentes en matière d'IA et d'interface.
  • Modélisation : définition des classes (plateau, joueur, IA, etc.) et structure de données.
  • Développement de l’IA : implémentation de plusieurs stratégies (ex : aléatoire, défensive, offensive).
  • Tests et débogage : vérification des comportements IA, détection des cas limites.
  • Implémentation de l’interface graphique avec JavaFX, en assurant une interaction fluide.
